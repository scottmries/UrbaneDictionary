# Urbane Dictionary

## Colloquialisms for the City-Dwelling Sophisticate

### Features

* Authentication
  * Twitter
  * Facebook
  * Email
* Submit definitions
* Attach media
  * Images
  * Vimeo/Youtube video
* Search definitions
* Vote on definitions

### Languages, Libraries, Etc.

* Ruby on Rails
* ReactJS
* Flux
* Gems
  * OmniAuth
  * Paperclip
  * AWS-sdk
  * Jbuilder
  * BCrypt
